F.A.Q.

# 运行服务器对系统有什么要求？

我们推荐运行 Linux 或 Windows (7/2008 R2+) 的 x86-64 系统、良好的上行带宽和最好的多核处理器。

# 有多少人可以加入我的服务器？

默认情况下，您的服务器最多支持 48 个使用 OneSync Infinity 的玩家。通过加入我们的元素俱乐部最多支持2048名玩家同时加入。

# 为什么我加入了 RedM 元素俱乐部，但仍然无法使用我的特权？

请等待一些时间来激活您的特权。 这最多可能需要 6 个小时。

# 什么是 OneSync 和 OneSync Infinity？

OneSync 及其扩展 OneSync Infinity 是我们建立在 Rockstar 原始网络代码之上的技术，可让您的服务器最多容纳 2048 名玩家。使用 OneSync，您可以使用其独特的服务器端游戏功能完全控制服务器上发生的事情。
