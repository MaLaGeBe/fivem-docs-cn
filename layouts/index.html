F.A.Q.

# 我会因为玩FiveM而被GTA:OL封号吗？

不会！FiveM不会与Rockstar在线服务进行交互，只会在您首次启动游戏副本时对其进行验证。此验证模拟了游戏的交互，Rockstar无法检测到。
即使下载服务器资源，它也根本不会修改您的游戏文件，因此您无需在FiveM或GTA:O之间进行任何移动。

# 为什么要询问我的登录信息？

当我们的自动验证失败时，我们需要检查您是否真正拥有GTAV。这些数据只会发送到Rockstar社交俱乐部服务，而不会传输给我们或任何其他方。

# 我可以在我的主机上玩 FiveM 吗？

不幸的是，在任何游戏机上提供FiveM都是不可行的，因为FiveM使用 高级互操作性机制仅在 PC 平台上可用。此外 FiveM依赖于PC平台对开发人员开放，而不需要任何 平台所有者的批准。

# OneSync是否需要我重新编写代码？

不，发布OneSync时，您所有现有的脚本仍可与游戏一起使用，并进行了微小的更改以支持更多的玩家。当然，OneSync将启用新的更好的方式来执行您在脚本中所做的某些事情，因此您可能希望在发布它们时加以利用。

# 我的FiveM崩溃了，我该怎么办？

你应该在社区发布你遇到的问题。请确保以 错误格式模板提供要求的所有信息，以便我们为您提供更好的帮助。

# M代表什么意思？

改进，而不是多人在线游戏。FiveM同时提供故事模式和多人在线游戏体验。这也允许玩家在故事模式中使用我们的一些功能。

# 我想搭建一个服务器，我该怎么做？

搭建服务器是很容易的，请参阅我们的服务器搭建页面。 遇到麻烦了？我们将在社区帮助你。不要犹豫，在论坛上发帖吧！

您还可以从我们推荐的服务器提供商处租用FiveM服务器并支持该项目。
